# To-Do List API Carajás

API REST em Node.js/TypeScript para gerenciar tarefas.  
Os dados ficam em memória (não tem banco de dados).

## Rotas

- `POST /tasks` : Adiciona tarefa
- `GET /tasks` : Lista tarefas
- `GET /tasks/:id` : Mostra tarefa por id
- `PUT /tasks/:id` : Atualiza tarefa
- `DELETE /tasks/:id` : Remove tarefa

## Observação

- Não consegui terminar a rota: `GET /tasks/metrics`

## Como rodar

- `npm install`
- `npm run dev`

Acesse em: `http://localhost:3000`
